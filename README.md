# MCBDD_Module_III
Module III Offline Activity

Note: to prepare this jupyter notebook was used Gemini Pro and availible information on Wiki.

Here is the Jupyter Notebook (.ipynb) outlining the relationships between the requested statistical tests. A key takeaway I found while putting this together is the mathematical equivalence between the Student's t-test and the Wald test when framed as a simple linear model. This highlights how traditional "t-tests" are simply special, constrained cases of Generalized Linear Models (GLMs).

Requirements: numpy, pandas, scipy.stats, and statsmodels.
